花粉识别系统使用说明
====================

一、系统要求
1. Windows操作系统
2. Python 3.8或更高版本（如未安装，请从 https://www.python.org/downloads/ 下载）
3. 至少4GB可用内存
4. 建议有独立显卡

二、文件说明
1. setup_and_run.bat - 自动部署和运行脚本
2. app_streamlit.py - 主程序
3. runs/train7/weights/best.pt - 模型文件（重要！）
4. requirements.txt - 依赖包列表

三、使用步骤
1. 确保所有文件在同一个文件夹中
2. 双击运行 setup_and_run.bat
3. 等待程序自动安装必要的包
4. 浏览器会自动打开应用界面

四、常见问题
1. 如果提示"未检测到Python"，请安装Python并将其添加到系统环境变量
2. 如果提示"模型文件未找到"，请确保模型文件在正确位置
3. 如果安装包时出错，请检查网络连接
4. 如果程序无法启动，请以管理员身份运行

五、注意事项
1. 首次运行时间可能较长，因为需要下载和安装必要的包
2. 请确保有稳定的网络连接
3. 建议在安装过程中不要进行其他操作
4. 如果之前安装过，可能需要先删除venv文件夹再重新运行
